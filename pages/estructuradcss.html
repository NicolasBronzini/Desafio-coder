Estructura CSS

La estructura de CSS se basa en selectores. Es decir que primero seleccionamos el objeto que queremos modificar, luego ponemos la propiedad, es decir de que forma queremos modificarlo. Al final se pone el valor que seria por ejemplo que color queremos que sea el fondo de la pagina.

En el desarrollo web, el modelo de caja CSS se refiere a cómo se modelan los elementos HTML en los motores de los navegadores y cómo las dimensiones de esos elementos HTML se derivan de las propiedades CSS.
